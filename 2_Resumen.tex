\bigskip
	\begin{center}
		\textcolor{black}{\textbf{RESUMEN}}	
	\end{center}
	\begin*
		\indent
	\end*
\textit{En este informe técnico se documenta el desarrollo de un sistema embebido de control, adquisición, supervisión, almacenamiento y telemetría satelital Iridium SBD para la boya instrumental de monitoreo acústico, en el marco del PIDDEF 0322.El sistema permite la operación autónoma de una plataforma marítima que adquiere variables ambientales, meteorológicas, de posición, energéticas y acústicas, las almacena localmente y transmite información seleccionada.Se basa en una computadora principal con software Python bajo GNU/Linux, organizada en módulos funcionales independientes con capa de bajo nivel y Máquina de Estados Finitos, coordinados por un proceso principal con enrutador de eventos y planificador de tareas.El documento describe la plataforma hardware, la arquitectura de software, interfaces, formatos de datos, mensajes binarios de telemetría y mecanismos de robustez y diagnóstico.La verificación incluyó 115 pruebas automatizadas sin hardware (75 \% de cobertura de líneas) y ensayos de integración en laboratorio que alcanzaron 118,7 h sin subsistema energético completo y 101,8 h con panel solar-batería.Se validó el procesamiento acústico, supervisión energética, telemetría y aislamiento ante fallas; la cadena Iridium verificó 90 mensajes recibidos y 85 payloads decodificados sin errores.
Los resultados confirman la validación funcional en laboratorio y sientan bases para el mantenimiento, evolución y ensayos en campo.}



\bigskip
	\begin{center}
		\textcolor{black}{\textbf{ABSTRACT}}
		\end{center}
			\begin*
			\indent
\textit{This technical report documents the development of an embedded system for control, acquisition, supervision, local storage and Iridium SBD satellite telemetry for the instrumented buoy for acoustic monitoring, within PIDDEF 0322.
The system enables autonomous operation of a maritime platform that acquires environmental, meteorological, position, power and acoustic variables, stores them locally and transmits selected information.
It is based on a main computer running Python software under GNU/Linux, organized into independent functional modules with low-level layer and Finite State Machine, coordinated by a main process with event router and central task scheduler.
The document describes the integrated hardware platform, software architecture, communication interfaces, data formats, binary telemetry messages and robustness/diagnostic mechanisms.
Verification included 115 automated hardware-free tests (75 \% line coverage) and laboratory integration trials that sustained 118.7 h without the complete power subsystem and 101.8 h with the solar panel–battery subsystem.
Acoustic processing, energy monitoring, telemetry and fault isolation were validated; the Iridium SBD chain confirmed 90 received messages and error-free decoding of 85 payloads.
The results provide functional laboratory integration validation and a technical foundation for maintenance, system evolution and future field autonomy trials.}   
	\end*
